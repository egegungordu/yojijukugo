# 四字熟語.com

毎日新しい四字熟語を紹介するサイトです。

## Getting Started

1. リポジトリをクローンします。
    ```bash
    git clone https://github.com/egegungordu/yojijukugo.com.git
    ```
2. 必要なパッケージをインストールします。
    ```bash
    npm install
    ```
3. 開発サーバーを起動します。
    ```bash
    npm run dev
    ```
